# 포트폴리오 기획서: Aurora Portfolio

## 📋 프로젝트 개요

**프로젝트명**: Aurora Portfolio
**목표**: 신비로운 오로라 테마의 인터랙티브 포트폴리오 웹사이트
**타겟**: 스타트업 프론트엔드 개발자 채용 담당자
**핵심 키워드**: 디자인 감각, 소통 능력, 인터랙티브 UX

### 1.1 테마

## 🏗️ 2. 페이지 구조

### 2.1 Hero 섹션
**컨셉**: "오로라 속을 항해하는 개발자"

**구성 요소**:
1. **배경 레이어**:
   - 깊은 남색 그라디언트 배경
   - CSS/Canvas로 구현한 오로라 효과

2. **인터랙티브 오로라**:
   - 마우스 위치에 따라 일렁이는 오로라
   - 부드러운 wave 애니메이션
   - 색상: 연보라 → 청록 → 핑크 그라디언트

3. **떠다니는 별 파티클**:
   - 작은 빛나는 별들이 화면을 가로지름
   - 오로라의 움직임에 영향받아 궤도 변경
   - CSS animation + JavaScript로 랜덤 생성

4. **헤드라인 (스토리텔링형)**:
   ```
   안녕하세요,
   디자인을 즐기는 프론트엔드 개발자
   [이름]입니다.

   즐겁게 만들고, 행복하게 전달합니다.
   ```
   - 타이핑 효과 애니메이션
   - Fade-in 순차 적용

5. **CTA 버튼**:
   - "프로젝트 보기" / "About Me"
   - 오로라 색상 테두리 애니메이션

**기술 스택**:
- React + MUI
- Canvas API (오로라 효과)
- Framer Motion (애니메이션)
- react-tsparticles (별 파티클)

### 2.2 About Me 섹션
**컨셉**: "나의 개발 여정"

**레이아웃**:
```
┌─────────────────────────────────────┐
│  타임라인 형식 (세로 스크롤)          │
│                                     │
│  ① 개발 시작 스토리                  │
│  ② 핵심 가치관                       │
│  ③ 개인적 매력 (취미/관심사)          │
│  ④ 성장 목표                         │
└─────────────────────────────────────┘
```

**상세 구성**:

**① 개발 시작 스토리** (스토리텔링 카드)
```
제목: "시작은 작은 웹사이트였습니다"

내용:
"나를 표현하는 웹사이트를 만들고 싶었습니다.
HTML 태그 하나하나가 신기했고,
CSS로 색을 입힐 때마다 즐거웠습니다.

그렇게 저는 코드로 디자인을 구현하는
프론트엔드 개발의 매력에 빠졌습니다."
```
- 일러스트: 노트북 앞의 실루엣
- 애니메이션: 스크롤 시 Fade-up

**② 핵심 가치관** (아이콘 카드)
```
제목: "즐겁게 디자인하기"

아이콘 3개:
🎨 "사용자를 생각한 디자인"
💬 "팀과의 원활한 소통"
✨ "끊임없는 개선"
```
- 호버 시 카드 확대
- 툴팁으로 상세 설명

**③ 개인적 매력** (프로필 카드)
```
좌측: 프로필 이미지 (오로라 테두리)
우측:
  취미: 줌바 💃
  관심사: AI 🤖

  "코딩 외의 시간에는..."
  간단한 소개 문구
```

**④ 성장 목표** (로드맵)
```
현재 → 목표

신입/주니어 개발자
    ↓
중급 개발자
    ↓
시니어 프론트엔드 개발자

"한 단계씩 성장하며,
더 나은 사용자 경험을 만들겠습니다."
```
- SVG 화살표 애니메이션

### 2.3 Skill Tree 섹션
**컨셉**: "Menu Board - 개발자의 기술 메뉴판"

**디자인**:
```
┌─────────────────────────────────────┐
│     🍽️ Developer's Menu 🍽️         │
│                                     │
│  [ Today's Special ]                │
│                                     │
│  HTML          ................ 下  │
│  "웹의 뼈대를 만듭니다"               │
│                                     │
│  CSS           ................ 下  │
│  "아름다운 스타일을 입힙니다"         │
│                                     │
│  JavaScript    ................ 下  │
│  "생명을 불어넣습니다"                │
│                                     │
│  React         ................ 下  │
│  "컴포넌트로 세상을 만듭니다"         │
│                                     │
│  Node.js       ................ 下  │
│  "서버와 대화합니다"                  │
│                                     │
│  * 숙련도: 上(상) 中(중) 下(하)      │
└─────────────────────────────────────┘
```

**인터랙티브 요소**:
1. **호버 효과**:
   - 메뉴 항목 호버 시 배경색 변경
   - 점선이 반짝이는 애니메이션

2. **클릭 효과**:
   - 클릭 시 해당 기술의 상세 설명 모달
   - 학습 경험, 프로젝트 적용 사례 표시

3. **스타일링**:
   - 빈티지한 메뉴판 느낌
   - 세리프 폰트 + 장식 요소
   - 연보라색 테두리

**숙련도 기준**:
- 下(하): 모든 기술 (현재 학습 중)
- 중장기 계획: 프로젝트 경험에 따라 업데이트

## 🛠️ 3. 기술 스택

### 3.1 Core
- **React 18** (Vite)
- **MUI** (Material-UI v6)
- **React Router**

### 3.2 Animation & Interaction
- **Framer Motion**: 페이지 전환, 스크롤 애니메이션
- **react-tsparticles**: 별 파티클
- **Canvas API**: 오로라 효과

### 3.3 Utilities
- **react-intersection-observer**: 스크롤 감지
- **react-type-animation**: 타이핑 효과

### 4.1 Breakpoints
- **Mobile**: 320px ~ 599px
- **Tablet**: 600px ~ 899px
- **Desktop**: 900px ~

### 4.2 섹션별 반응형 처리

- Mobile: 세로 레이아웃, 작은 오로라

- Mobile: 카드 세로 나열
- Desktop: 타임라인 형식

**Skill Tree**:
- Mobile: 1열 메뉴판
- Desktop: 2열 메뉴판

**Projects**:
- Mobile: 1열 Grid
- Tablet: 2열 Grid
- Desktop: 3열 Grid
